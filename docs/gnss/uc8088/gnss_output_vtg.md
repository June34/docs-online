# VTG
![](.\picture\vtg.png)

VTG航迹及对地速度各个字段含义如下所示：

1.真航角，单位：度

2.T-真航角

3.磁偏角，单位：度

4.M-磁偏角

5.对地速度，单位：节

6.N-节

7.对地速度，单位：千米/小时

8.K-千米/小时

9.FAA模式指示器

10.校验和



示例说明

$GPVTG,0.00,T,,M,0.004,N,0.007,K,A*3E

1.真航角 - 0.00度

2.真航角

3.磁偏角 - 无信息

4.磁偏角

5.对地速度 - 0.004节

6.节

7.对地速度 - 0.007千米/小时

8.千米/小时

9.FAA模式指示器 - 自动
